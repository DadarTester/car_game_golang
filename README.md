# car_Racing_Game_Go
## Running the Game
Having Go installed, you can run ```<go run>``` . from the root directory to play the game.

## Controls
* Left/Right arrow - move aside
* Up arrow - Speed up
* Down arrow - Speed down
* Space - Restart the game
* Esc - exit
  
## Stack
* Golang 1.2.4
* [Ebitengine] engine
* All sprites made by myself
